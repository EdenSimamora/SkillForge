# 🚀 SkillForge

SkillForge is a Django-based web application for tracking and managing personal skills with progress visualization, authentication system, and multi-language support (English & Indonesian).

---

## ✨ Features

- 🔐 User Authentication (Login/Register)
- 👤 Multi-user Skill Management
- 📊 Skill Progress Tracking
- 🌐 Multi-language Support (EN / ID)
- 🎨 Bootstrap UI + Font Awesome
- 🧠 Django Admin Panel
- 📱 Responsive Design

---

## 🛠️ Tech Stack

- Python 3.13+
- Django 6+
- Bootstrap 5
- SQLite3
- HTML / CSS / JS

## 🚀 Installation

```bash
git clone https://github.com/USERNAME/SkillForge.git
cd SkillForge

python -m venv venv
venv\Scripts\activate  # Windows

pip install -r requirements.txt

python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```
